Robot Framework Native UI Testing

ฝึกฝนการเขียน Keyword-driven Testing ด้วย Robot Framework ครอบคลุมการประกาศ Settings, Variables, Custom Keywords, macOS Native UI Automation ผ่านแอป Kouen จริง (ไม่ใช้ Appium) และ OS Processes

หมายเหตุ: โครงสร้าง RF และหลักคิด (locator ที่เสถียร, test isolation, retry ให้ถูกจุด) ที่ได้จากบทเรียนนี้ต่อยอดไปสาย Appium Mobile Native และ Appium Flutter ได้จริง แต่ต้องไป apply เรียนรู้ library/locator เฉพาะแพลตฟอร์มเพิ่มเติมทุกครั้งที่ข้ามสาย — เหมือนคนเขียน Selenium เป็นแล้วต้อง apply เพิ่มตอนย้ายไปเขียน Appium Flutter เช่นกัน
